# EduNexus 知识库本地存储 - 快速入门

## 5分钟快速上手

### 第一步：打开知识库页面

访问 `/kb` 路由，你会看到知识库界面。

### 第二步：创建知识库

1. 点击左上角的"选择知识库"按钮
2. 在弹出的对话框中点击"新建知识库"
3. 输入知识库名称（例如："我的笔记"）
4. 可选：输入存储路径（留空则使用默认路径）
5. 点击"创建"

### 第三步：创建文档

1. 点击工具栏的"新建"按钮
2. 开始编写你的第一篇文档
3. 使用 Markdown 语法编辑内容
4. 点击"保存"按钮保存文档

### 第四步：使用高级功能

#### 双链笔记
在文档中使用 `[[文档名]]` 创建链接：
```markdown
参考 [[欢迎文档]] 了解更多信息。
```

#### 标签系统
使用 `#标签` 组织内容：
```markdown
这是一篇关于 #React 和 #TypeScript 的笔记。
```

#### 导入现有文档
1. 点击"导入"按钮
2. 选择一个或多个 .md 文件
3. 文档会自动导入到当前知识库

#### 导出文档
1. 选择要导出的文档
2. 点击"导出"按钮
3. 文档会以 .md 格式下载到本地

## 常用快捷键

- `Ctrl + S`: 保存文档
- `Ctrl + N`: 新建文档
- `Ctrl + E`: 切换编辑/预览模式

## 界面说明

### 左侧面板
- **知识库选择器**：切换不同的知识库
- **搜索框**：搜索文档标题和内容
- **标签筛选**：按标签过滤文档
- **文档列表**：显示所有文档或最近文档

### 中间面板
- **工具栏**：新建、保存、导出、导入、删除等操作
- **编辑器**：Markdown 编辑器
- **预览区**：实时预览渲染效果

### 右侧面板
- **大纲**：文档标题结构
- **反向链接**：引用当前文档的其他文档
- **标签云**：当前文档的所有标签

## 最佳实践

### 1. 文档命名
- 使用清晰、描述性的标题
- 避免使用特殊字符
- 保持标题简洁（建议不超过50个字符）

### 2. 标签使用
- 每个文档建议使用 2-5 个标签
- 使用一致的标签命名规范
- 定期整理和合并相似标签

### 3. 双链笔记
- 在文档间建立有意义的连接
- 使用双链创建知识网络
- 定期查看反向链接发现新的关联

### 4. 文档组织
- 按主题创建不同的知识库
- 使用标签进行横向分类
- 利用双链建立纵向关联

### 5. 备份策略
- 定期导出重要文档
- 考虑使用多个知识库分散风险
- 重要内容建议同时保存到云端

## 示例：创建一个学习笔记系统

### 1. 创建知识库
```
知识库名称：学习笔记
路径：~/Documents/EduNexus/学习笔记
```

### 2. 创建索引文档
```markdown
# 学习笔记索引

## 编程语言
- [[JavaScript 基础]]
- [[TypeScript 进阶]]
- [[Python 入门]]

## 框架和库
- [[React 学习路线]]
- [[Next.js 实战]]
- [[Tailwind CSS 技巧]]

## 计算机科学
- [[数据结构与算法]]
- [[设计模式]]
- [[系统设计]]

#索引 #学习
```

### 3. 创建具体笔记
```markdown
# React 学习路线

## 基础概念
- 组件
- Props 和 State
- 生命周期

## Hooks
- useState
- useEffect
- useContext
- 自定义 Hooks

## 高级主题
- 性能优化
- 代码分割
- SSR/SSG

参考：[[JavaScript 基础]]、[[TypeScript 进阶]]

#React #前端 #学习
```

### 4. 建立知识网络
随着笔记增多，文档间会自动形成网络：
- 通过双链连接相关概念
- 通过标签组织同类内容
- 通过反向链接发现知识关联

## 故障排除

### 问题：找不到"选择知识库"按钮
**解决方案**：刷新页面，确保组件已正确加载。

### 问题：保存失败
**解决方案**：
1. 检查浏览器控制台是否有错误
2. 确认浏览器支持 IndexedDB
3. 检查存储空间是否充足

### 问题：导入文件失败
**解决方案**：
1. 确保文件格式为 .md
2. 检查文件编码为 UTF-8
3. 确保文件大小合理（建议 < 10MB）

### 问题：搜索结果不准确
**解决方案**：
1. 使用更具体的关键词
2. 尝试使用标签筛选
3. 检查文档内容是否包含搜索词
