# 全局 AI 助手使用指南

## 概述

全局 AI 助手是一个可拖拽、可调整大小的悬浮窗口，提供智能对话功能。

## 主要功能

### 1. 打开/关闭

- **快捷键**: `Cmd/Ctrl + K`
- **点击按钮**: 点击右下角的紫色悬浮按钮
- **关闭**: 点击窗口右上角的 X 按钮

### 2. 拖拽移动

- 点击并拖拽窗口顶部的紫色标题栏
- 窗口会自动限制在屏幕范围内
- 位置会自动保存到 localStorage

### 3. 调整大小

- 拖拽窗口右下角的调整大小手柄
- 最小尺寸: 320×400 像素
- 最大尺寸: 800×900 像素
- 调整时会显示当前尺寸提示
- 尺寸会自动保存到 localStorage

### 4. 窗口控制

#### 标题栏按钮（从左到右）：

1. **重置布局** (⛶ 图标)
   - 重置窗口位置和尺寸到默认值
   - 会刷新页面以应用更改

2. **重新开始** (↻ 图标)
   - 清空所有对话消息
   - 重置输入框

3. **最小化** (− 图标)
   - 折叠窗口内容，只保留标题栏
   - 再次点击可展开

4. **关闭** (× 图标)
   - 关闭助手窗口
   - 可通过快捷键或悬浮按钮重新打开

### 5. 对话功能

#### 快速操作
- 首次打开时显示快速操作按钮
- 点击按钮可快速填充常用提示词

#### 发送消息
- 在输入框中输入消息
- 按 `Enter` 发送
- 按 `Shift + Enter` 换行

#### 消息操作
- **复制**: 点击助手消息下方的复制按钮
- **清空对话**: 点击输入框上方的"清空对话"按钮

## 技术特性

### z-index 层级
- 助手窗口: `z-index: 40`
- 悬浮按钮: `z-index: 40`
- 不会遮挡模态框（通常 z-index: 50+）和下拉菜单

### 数据持久化
- 窗口位置保存在 `localStorage['ai-assistant-position']`
- 窗口尺寸保存在 `localStorage['ai-assistant-size']`
- 关闭浏览器后重新打开会恢复上次的位置和尺寸

### 响应式设计
- 支持深色模式
- 平滑的动画过渡
- 触觉反馈（悬停、点击效果）

## 视觉设计

### 颜色方案
- **主色调**: 紫色到粉色渐变 (`from-purple-500 to-pink-500`)
- **背景**: 白色 / 深灰色（深色模式）
- **边框**: 灰色 200 / 700（深色模式）
- **阴影**: 2xl 级别，提供良好的层次感

### 交互反馈
- 拖拽时窗口半透明（opacity: 0.9）
- 调整大小时禁用文本选择
- 悬停时按钮高亮
- 加载时显示动画点

## 使用建议

1. **首次使用**: 将窗口拖拽到你习惯的位置，调整到合适的大小
2. **快速访问**: 记住快捷键 `Cmd/Ctrl + K`，随时唤起助手
3. **多任务**: 最小化窗口可以保持助手在后台，不占用屏幕空间
4. **重置**: 如果窗口位置或大小不理想，使用重置布局功能

## 故障排除

### 窗口位置异常
- 点击"重置布局"按钮恢复默认位置

### 窗口太小/太大
- 拖拽右下角调整到合适大小
- 或使用"重置布局"恢复默认尺寸

### 快捷键不工作
- 确保没有其他应用占用 `Cmd/Ctrl + K`
- 尝试点击悬浮按钮打开

### 对话无响应
- 检查网络连接
- 刷新页面重试
- 查看浏览器控制台是否有错误信息
