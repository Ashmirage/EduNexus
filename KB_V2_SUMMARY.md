# 知识库 V2 - 全新设计总结

**创建时间**: 2026-03-11
**版本**: v2.0.0
**状态**: ✅ 已完成

---

## 🎯 设计目标

根据用户反馈，原知识库存在以下问题：
1. UI 流畅度不够
2. 右侧栏设计不合理
3. AI 思维导图生成后无法查看
4. 整体设计不如成熟的笔记应用

**解决方案**：参考 Notion、飞书文档、语雀等优秀笔记应用，完全重新设计知识库界面。

---

## ✨ 核心改进

### 1. 布局优化

**旧版本（V1）**：
- 混乱的多面板布局
- 右侧栏功能堆砌
- 缺乏清晰的信息层次

**新版本（V2）**：
- **左侧栏（280px，可折叠）**：
  - 知识库切换器
  - 全局搜索
  - 文档树（支持拖拽、嵌套）
  - 快速访问（最近、收藏、标签）

- **中间编辑区（flex-1）**：
  - 简洁的顶部工具栏
  - 大字体标题区
  - Tiptap 富文本编辑器
  - 底部状态栏

- **右侧面板（320px，可折叠）**：
  - 标签页组织：大纲、AI 助手、属性、历史
  - 每个功能独立展示
  - 支持全屏查看

### 2. 编辑器升级

**旧版本（V1）**：
- 简单的 Textarea
- 功能有限
- 体验一般

**新版本（V2）**：
- **Tiptap 富文本编辑器**（基于 ProseMirror）
- 完整 Markdown 支持
- 支持的格式：
  - 标题（H1-H6）
  - 粗体、斜体、删除线、代码
  - 有序列表、无序列表、任务列表
  - 引用块、代码块
  - 表格、图片、链接
  - 水平分割线
- 实时预览
- 自动保存（防抖 1 秒）

### 3. AI 功能优化

**旧版本（V1）**：
- AI 思维导图显示不完整
- 功能混在一起
- 交互不友好

**新版本（V2）**：

#### AI 思维导图
- 使用 **ReactFlow** 可视化
- 支持缩放、平移
- 节点可点击展开
- 支持导出为图片
- 可以全屏查看
- 自动布局优化

#### AI 摘要
- 卡片式展示
- 一键生成
- 支持复制
- 支持插入到文档

#### AI 问答
- 对话式界面
- 支持上下文
- 支持引用文档内容
- 流式输出

### 4. 交互优化

**新增功能**：
- ✅ 流畅的动画过渡（Framer Motion）
- ✅ 丰富的快捷键支持
- ✅ 拖拽排序
- ✅ 右键菜单
- ✅ 悬停预览
- ✅ 自动保存提示
- ✅ 保存状态指示

**快捷键列表**：
```
Cmd/Ctrl + S     - 保存文档
Cmd/Ctrl + K     - 全局搜索
Cmd/Ctrl + B     - 粗体
Cmd/Ctrl + I     - 斜体
Cmd/Ctrl + U     - 下划线
Cmd/Ctrl + E     - 代码
Cmd/Ctrl + [     - 折叠左侧栏
Cmd/Ctrl + ]     - 折叠右侧栏
Cmd/Ctrl + \     - 切换预览
Cmd/Ctrl + /     - 显示快捷键
```

## 📦 技术实现

### 核心组件（8 个）

1. **kb-layout.tsx** - 三栏布局容器
   - 可折叠侧边栏
   - 响应式设计
   - 快捷键支持

2. **kb-sidebar.tsx** - 左侧边栏
   - 知识库切换
   - 搜索框
   - 文档树
   - 快速访问

3. **kb-editor.tsx** - Tiptap 编辑器
   - 富文本编辑
   - Markdown 支持
   - 自动保存

4. **editor-toolbar.tsx** - 编辑器工具栏
   - 格式化按钮
   - 插入功能
   - 悬停提示

5. **kb-right-panel.tsx** - 右侧面板
   - 标签页切换
   - 大纲导航
   - AI 功能
   - 文档属性

6. **ai-mindmap.tsx** - AI 思维导图
   - ReactFlow 可视化
   - 缩放、平移
   - 导出功能

7. **ai-summary.tsx** - AI 摘要
   - 一键生成
   - 卡片展示
   - 复制、插入

8. **ai-chat.tsx** - AI 问答
   - 对话界面
   - 流式输出
   - 上下文支持

### API 路由（3 个）

- `/api/kb/mindmap` - 思维导图生成
- `/api/kb/summary` - 文档摘要生成
- `/api/kb/chat` - AI 问答

### 技术栈

- **编辑器**: Tiptap (ProseMirror)
- **可视化**: ReactFlow
- **动画**: Framer Motion
- **UI**: Radix UI + Tailwind CSS
- **AI**: LangChain + OpenAI
- **存储**: IndexedDB

## 🚀 使用指南

### 1. 启动开发服务器

```bash
# 方式 1: 使用 npm
cd apps/web
npm run dev

# 方式 2: 使用启动脚本
./start-kb-v2.sh    # Linux/Mac
start-kb-v2.bat     # Windows
```

### 2. 访问新版本

```
http://localhost:3000/kb-v2
```

### 3. 配置 AI 功能

创建 `.env.local` 文件：

```env
OPENAI_API_KEY=your_api_key_here
OPENAI_API_BASE=https://api.openai.com/v1
```

### 4. 基本操作

#### 创建文档
1. 点击左侧栏的 "新建文档" 按钮
2. 输入标题
3. 开始编辑内容

#### 使用工具栏
- 点击工具栏按钮格式化文本
- 悬停查看快捷键提示

#### 使用 AI 功能
1. 切换到右侧面板的 "AI 助手" 标签
2. 选择功能：思维导图、摘要、问答
3. 点击生成按钮

#### 快捷键
- 按 `Cmd/Ctrl + /` 查看所有快捷键
