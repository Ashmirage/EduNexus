# 全局 AI 助手 - 快速参考

## 🎯 核心功能一览

### 打开助手
```
快捷键: Cmd/Ctrl + K
或点击右下角紫色悬浮按钮
```

### 窗口控制
```
┌─────────────────────────────────────┐
│ ≡ ✨ AI 助手    [⛶] [↻] [−] [×]  │ ← 标题栏（可拖拽）
├─────────────────────────────────────┤
│                                     │
│         对话内容区域                 │
│                                     │
├─────────────────────────────────────┤
│ [输入框]              [发送]        │
└─────────────────────────────────┘◢  │ ← 调整大小手柄
```

### 按钮功能
- **⛶ 重置布局**: 恢复默认位置和尺寸
- **↻ 重新开始**: 清空所有对话
- **− 最小化**: 折叠窗口内容
- **× 关闭**: 关闭助手窗口

## 🎨 视觉特性

### 颜色方案
```
主色调: 紫色 → 粉色渐变
背景:   白色 / 深灰（深色模式）
边框:   灰色 200 / 700
阴影:   2xl 级别
```

### 尺寸范围
```
默认: 420 × 620 像素
最小: 320 × 400 像素
最大: 800 × 900 像素
```

### z-index 层级
```
助手窗口: 40
悬浮按钮: 40
模态框:   50+
下拉菜单: 50+
```

| 操作 | 功能 |

## 🔧 故障排除

### 窗口位置异常
```
解决: 点击 [⛶] 重置布局
```

### 窗口太小/太大
```
解决: 拖拽右下角调整
或点击 [⛶] 重置布局
```

### 快捷键不工作
```
解决: 点击悬浮按钮打开
```

### 对话无响应
```
1. 检查网络连接
2. 刷新页面重试
3. 查看控制台错误
```
