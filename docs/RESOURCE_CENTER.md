# 资源中心系统

资源中心是 EduNexus 的核心功能之一，帮助用户管理和分享学习资源，构建个人知识库。

## 功能特性

### 1. 资源管理

- **多种资源类型**
  - 📄 文档：PDF、Word、PPT 等
  - 🎥 视频：YouTube、Bilibili、本地视频
  - 🔧 工具：开发工具、在线服务
  - 🌐 网站：学习网站、博客文章
  - 📚 书籍：电子书、在线教材

- **资源上传**
  - 支持文件上传（最大 100MB）
  - 支持添加外部链接
  - 自动提取元数据

- **资源元数据**
  - 标题和描述
  - 标签分类
  - 作者和来源
  - 统计信息（浏览数、收藏数、评分）

### 2. 资源搜索和筛选

- **关键词搜索**
  - 搜索标题、描述、标签
  - 实时搜索结果

- **高级筛选**
  - 按资源类型筛选
  - 按标签筛选（多标签支持）
  - 按状态筛选

- **排序选项**
  - 最新添加
  - 浏览最多
  - 收藏最多
  - 评分最高

### 3. 收藏夹管理

- **创建收藏夹**
  - 自定义名称和描述
  - 选择颜色和图标
  - 公开或私密设置

- **收藏夹操作**
  - 编辑收藏夹信息
  - 分享收藏夹（生成分享链接）
  - 导出为 Markdown 格式
  - 删除收藏夹

- **资源收藏**
  - 一键收藏资源
  - 添加到指定收藏夹
  - 为收藏添加笔记
  - 评分（1-5 星）

### 4. 资源预览

- **PDF 预览**
  - 在线预览 PDF 文档
  - 无需下载

- **图片预览**
  - 高清图片展示
  - 缩放和查看

- **视频播放**
  - YouTube 视频嵌入
  - Bilibili 视频嵌入
  - 本地视频播放

- **网站预览**
  - iframe 嵌入预览
  - 安全沙箱模式

### 5. 资源推荐

- **AI 个性化推荐**
  - 基于用户收藏历史
  - 基于学习路径
  - 基于知识点匹配

- **热门资源**
  - 社区热门排行
  - 综合评分排序

- **相似资源**
  - 基于标签匹配
  - 基于类型匹配
  - 基于作者和来源

- **基于上下文推荐**
  - 当前学习主题相关
  - 学习路径相关
  - 知识节点相关

## 使用场景

### 学习资源整理

1. 添加学习资源（文档、视频、网站）
2. 使用标签分类
3. 创建主题收藏夹
4. 添加学习笔记

### 知识库构建

1. 收集各类学习资料
2. 按主题组织收藏夹
3. 添加评分和笔记
4. 导出为 Markdown

### 资源分享

1. 创建公开收藏夹
2. 生成分享链接
3. 分享给同学或团队
4. 协作学习

### 学习路径规划

1. 根据学习目标收集资源
2. 按难度和顺序组织
3. 跟踪学习进度
4. 获取 AI 推荐

## 技术实现

### 前端

- **React + TypeScript**
- **Tailwind CSS** - 样式系统
- **Lucide Icons** - 图标库
- **shadcn/ui** - UI 组件库

### 数据存储

- **LocalStorage** - 客户端存储
- **类型安全** - TypeScript 类型定义
- **数据持久化** - JSON 格式

### 推荐算法

- **协同过滤** - 基于用户行为
- **内容过滤** - 基于标签和类型
- **混合推荐** - 综合多种策略
- **上下文感知** - 考虑学习场景

```
│       └── bookmark-manager.tsx        # 收藏夹管理

    └── resources/
        ├── resource-types.ts           # 类型定义

        └── sample-data.ts              # 示例数据
```
