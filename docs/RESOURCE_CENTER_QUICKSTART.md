# 资源中心快速启动指南

## 快速开始

### 1. 访问资源中心

在侧边栏点击 "📦 资源中心" 或直接访问 `/resources`

### 2. 查看示例数据

首次访问时，系统会自动生成示例数据：
- 15+ 个学习资源（Python、JavaScript、机器学习等）
- 3 个收藏夹（Python 学习、前端开发、机器学习）
- 多个收藏和评分

### 3. 添加资源

点击右上角 "添加资源" 按钮：

**添加链接：**
1. 选择 "添加链接"
2. 选择资源类型（文档/视频/工具/网站/书籍）
3. 输入资源链接
4. 填写标题和描述
5. 添加标签（用逗号分隔）
6. 可选：填写作者和来源
7. 点击 "添加资源"

**上传文件：**
1. 选择 "上传文件"
2. 选择资源类型
3. 拖拽或点击上传文件
4. 填写其他信息
5. 点击 "添加资源"

### 4. 搜索和筛选

**关键词搜索：**
- 在搜索框输入关键词
- 按 Enter 或点击 "搜索" 按钮

**高级筛选：**
- 选择资源类型（全部/文档/视频/工具/网站/书籍）
- 选择排序方式（最新/浏览最多/收藏最多/评分最高）
- 点击 "标签筛选" 选择多个标签

**清除筛选：**
- 点击 "清除" 按钮重置所有筛选条件

### 5. 收藏资源

**快速收藏：**
1. 在资源卡片上点击 "收藏" 图标
2. 资源自动添加到 "未分类"

**添加到收藏夹：**
1. 点击资源卡片的 "更多" 菜单
2. 选择 "添加到收藏夹"
3. 选择目标收藏夹

**评分和笔记：**
1. 收藏资源后
2. 在 "我的收藏" 标签页找到资源
3. 点击资源卡片
4. 添加评分（1-5 星）
5. 添加学习笔记

### 6. 管理收藏夹

**创建收藏夹：**
1. 在左侧栏点击 "新建收藏夹"
2. 输入名称和描述
3. 选择颜色
4. 点击 "创建"

**编辑收藏夹：**
1. 点击收藏夹的 "更多" 菜单
2. 选择 "编辑"
3. 修改信息
4. 点击 "保存"

**分享收藏夹：**
1. 点击收藏夹的 "更多" 菜单
2. 选择 "分享"
3. 分享链接自动复制到剪贴板

**导出收藏夹：**
1. 点击收藏夹的 "更多" 菜单
2. 选择 "导出"
3. 下载 Markdown 格式文件

### 7. 查看推荐

切换到 "推荐" 标签页：

**为你推荐：**
- 基于你的收藏历史
- 基于你的学习偏好
- AI 个性化推荐

**社区热门：**
- 收藏最多的资源
- 评分最高的资源
- 浏览最多的资源

## 使用技巧

### 标签管理

- 使用一致的标签命名（如：Python、JavaScript）
- 使用多个标签增加可发现性
- 标签支持中英文

### 资源组织

- 按主题创建收藏夹（如：Python 学习、前端开发）
- 按项目创建收藏夹（如：毕业设计、工作项目）
- 按难度创建收藏夹（如：入门、进阶、高级）

### 学习笔记

- 记录资源的重点内容
- 记录学习心得
- 记录待办事项

### 评分系统

- 1 星：不推荐
- 2 星：一般
- 3 星：还可以
- 4 星：推荐
- 5 星：强烈推荐

## 常见问题

**Q: 如何删除资源？**
A: 点击资源卡片的 "更多" 菜单，选择 "删除"

**Q: 如何取消收藏？**
A: 点击资源卡片上的 "已收藏" 图标

**Q: 收藏夹可以嵌套吗？**
A: 当前版本不支持嵌套，但可以使用标签来组织

**Q: 如何批量导入资源？**
A: 当前版本需要逐个添加，未来会支持批量导入

**Q: 数据存储在哪里？**
A: 数据存储在浏览器的 LocalStorage 中

**Q: 如何备份数据？**
A: 可以导出收藏夹为 Markdown 格式

## 键盘快捷键

- `Ctrl/Cmd + K` - 聚焦搜索框（计划中）
- `Esc` - 关闭对话框
- `Enter` - 提交表单
