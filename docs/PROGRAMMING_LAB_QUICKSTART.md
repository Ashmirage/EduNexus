# 编程实验室快速入门

## 快速开始

### 1. 访问实验室
1. 打开 EduNexus
2. 进入"学习工作区"
3. 点击右侧"实验室"标签

### 2. 创建第一个项目
1. 点击"新建项目"按钮
2. 输入项目名称（例如："我的第一个项目"）
3. 选择模板（推荐：Python 项目）
4. 点击"创建项目"

### 3. 编辑代码
1. 在左侧文件树中选择文件（例如：main.py）
2. 在编辑器中修改代码
3. 代码会自动保存

### 4. 运行代码
1. 切换到"执行器"标签
2. 点击"运行代码"按钮
3. 查看输出结果

## 常用操作

### 文件操作
- **新建文件**: 点击文件夹旁的 ➕ 图标
- **新建文件夹**: 点击 📁 图标
- **重命名**: 点击 ✏️ 图标
- **删除**: 点击 🗑️ 图标

### 编辑器操作
- **切换语言**: 使用顶部语言选择器
- **切换主题**: 选择浅色/深色/高对比度
- **调整字体**: 选择 12-20px
- **格式化代码**: 点击"格式化"按钮

### 项目操作
- **切换项目**: 使用项目选择器
- **导出项目**: 点击 ⬇️ 图标
- **导入项目**: 点击 ⬆️ 图标
- **删除项目**: 点击 🗑️ 图标

## 常见问题

**Q: Python 代码运行很慢？**
A: 首次运行需要加载 Python 环境，之后会很快。

**Q: 如何保存项目？**
A: 项目自动保存到浏览器，也可以导出为 JSON 文件。

**Q: 支持哪些编程语言？**
A: 目前支持 JavaScript、TypeScript 和 Python 执行。

**Q: 代码执行有时间限制吗？**
A: 是的，为了安全，代码执行限制为 5 秒。
